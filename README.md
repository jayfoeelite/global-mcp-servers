# Global MCP Servers

This repository contains a collection of MCP servers, each with a specific purpose.

## MCP Servers

- **doc-fetcher-mcp**: Fetches documentation from a given URL.
- **memory-bank-mcp**: Provides a persistent memory for the AI assistant.
- **roo-code-memory-bank**: A memory system integrated with VS Code to maintain context across sessions.
- **sequentialthinking**: A collection of MCP servers for various tasks, including memory, file system, git, GitHub, and Postgres.
